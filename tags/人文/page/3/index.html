世界杯：越位了？被吹掉的首个进球背后的科技

文章来自微信公众号“科文路”，欢迎关注、互动。转发须注明出处。

在凌晨 0 点进行的世界杯首场比赛，卡塔尔对阵厄瓜多尔，中，第一个进球在开场几分钟后由厄瓜多尔队创造，但随后被裁判取消。原因是“越位”。正在大伙儿感叹卡塔尔的“钞”能力时，视频回放了视频助理裁判的判定。确定无误。

助力这次判罚的技术被称为“半自动越位识别技术”，是 VAR 和“科技足球”的结合。

这一技术利用安装在球场屋顶下方的 12 个专用摄像头，追踪场上的足球和每一名球员身上的 29 个点位，每秒可记录 50 次，从而精准算出他们在场地内的准确位置。——《IT之家》
