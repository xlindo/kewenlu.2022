荐歌：花房姑娘（崔健）

文章来自微信公众号“科文路”，欢迎关注、互动。转发须注明出处。

端午愉快~

本周荐歌《花房姑娘》，来自崔健。

经典就是经典，这首发表于 1989 年的摇滚乐现在听也完全没有一点过时的感觉。

韩磊在《歌手》上的翻唱也值得流传。

创作背景：由于客观环境条件，创作《花房姑娘》时，必须进入录音棚一次性完成，但第一次录完了以后崔健团队的人都发现该曲特别糟糕，后面都是空的。大家确定先吃饭，崔健没有跟去，而是让其他人给他带个包子。于是他开始一个人录，一边听一边想办法。当其他人回来的时候，崔健已经把完整的歌曲以及伴唱写出来了。崔健当时喜欢一个女孩，他写过一首诗和一首曲子，后来他觉得曲子的词不太合适，于是进行了缩减，便有了《花房姑娘》 。
