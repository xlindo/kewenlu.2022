少折腾，给Ubuntu 20.04 LTS安个优麒麟搜狗输入法

折腾了好几个小时，官方搜狗输入法也没装好，fcitx5 也没装好。跟 wine 微信一样，最后转投了优麒麟版的怀抱。安装简洁，一次成功。

以下内容大部分来自后文，点击 “阅读原文” 可跳转至 Ubuntu 20.04 LTS 安装搜狗输入法，只需三条命令，还能自动更新 - 小镜湖 [1]

Ubuntu 20.04 已经发布了快一个月了，一直没能顺利地安装搜狗输入法。原因也很简单，就是搜狗输入法的依赖 (qt4 相关) 在 20.04 中被去掉了，而且无法直接修复。没有办法，用了两个星期系统自带的输入法，其实还是不错的，不过相比做了深耕输入法多年的搜狗，是比不上的。今天再次尝试安装了一下搜狗输入法，发现还是可以实现的，而且方法并不复杂！

添加 ubuntukylin 源

参考 UKUI 技术 [2]，添加源。源中就包含了麒麟版的搜狗输入法。同时我们也能够及时和上游保持更新。

使用 apt 安装搜狗输入法

接着就能够直接安装软件包了，注意包名是 sogouimebs 而不是原来的 sogoupinyin 了。这个版本完全可以在 20.04 上运行，但是在其他地方暂时是找不到的，因为是麒麟社区定制的版本。但是由于 ubuntu 和优麒麟一脉相承的关系，我们可以直接拿来用。

sudo apt install sogouimebs

安装完成后，使用 ctrl + 空格 就可以看到搜狗输入法熟悉的游标啦。

配置界面同样也可以通过在终端中输入 sogouIme-configtool 打开。如图

设置默认输入法

最后需要把默认输入法设置为 fcitx，重启电脑就可以使用搜狗输入法了。现在就可以体验了 windows 下相同的打字体验了。

禁用 IBus(可选)

经过一天多的使用，我发现偶尔遇到输入卡住的情况，比如重启 fcitx 才能解决。幸好及时看到知乎网友的评论，卸载 IBus 或许能解决这个问题 [3]。ibus 和 fcitx 发生冲突不是新鲜问题了，如果有相同的问题不妨卸载 ibus 试一试。

或者简单粗暴，直接卸载了 ibus

sudo apt remove ibus

再补充：即使是卸载 ibus 这一步也出现了问题，因为 zoom 竟然依赖 ibus，而 zoom 肯定是不能卸掉的。幸好在 debian wiki 上找到了禁用 ibus 的方法 [5]。相比直接改名会更加稳定。
