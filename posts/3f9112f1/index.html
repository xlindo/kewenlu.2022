工具：笔记神器Obsidian发布1.0版

文章来自微信公众号“科文路”，欢迎关注、互动。转发须注明出处。

Obsidian的官方标语是“你永远的第二大脑”。实际使用后，这款笔记（知识库）工具强大的功能也的确配得上这样的描述。

于是，它搭配着 git 已经成为了我的主力笔记工具。对个人使用免费，更值得推荐！（即便后续不免费了，原生的markdown格式也能迅速迁移）

最近，这款软件在漫长的大版本为 0 的时代后，发布了 1.0 版。正如更新日志所说，“我们已经能足够自信的扔掉 beta 的标签”。

更新

主要的更新有：

  • 多标签页，终于有多标签了，可以在多个笔记间快速切换了
  • 标签组，这个感觉没多大用
  • 新的外观设计，不知道是不是适配的问题，我立马切到了这个默认的新外观，看着比原来舒服太多

其他的更新有：

  • 主题商店的改进。现在其支持查看主题的版本历史，就像插件商店一样。需要注意的是，与 v0.16 不兼容的主题已被标记为“legacy”，这些主题在更新之前无法被安装
  • 在文件列表中增加了折叠所有的选项，该选项适用于文件列表和标签列表
  • 增加了折叠与展开命令，这两个命令基于当前光标位置生效
  • 增加了更换主题命令，从而使得你可以快速切换主题
  • 自动生成的标题链接现在可以保留更多标点符号，以便你能更好地理解上下文
  • 在 macOS 菜单中增加了查找与替换、导出 PDF、分享、格式化文本等选项
  • 添加了 Shift + Up/Down 快捷键。该快捷键用于搜索结果为焦点时展开搜索结果
  • 重命名了垂直拆分和水平拆分命令，以方便你更好的理解命令作用
  • 现在在列表中按下 home 键将自动定位列表开头
  • 官方同步服务：版本历史页面中增加了一个开关，使你可以查看当前文件某个版本的内容，或是当前版本与其他版本的差异
  • 现在 Obsidian 更新后将自动在软件内显示更新记录
  • 改善了插件商店和主题插件介绍页的图片，并使介绍页支持视频
  • 修改了许多命令中的措辞，以减少模糊性
  • 提升了半透明模式下的软件性能
  • 现在避免了软换行错误对齐缩进的问题
  • 在导出为 PDF 的页面中增加了一个选项，从而让你选择是否将笔记标题显示在文档开头
  • 发布服务：未发布和未修改页面现在默认折叠
  • 修复了检索式包含 “path:” 有时会使搜索结果重复的问题
  • 修复了多行 query 无法工作的问题
  • 修复了加载系统自带字体时内存泄漏的问题
  • 修复文件列表丢失排序顺序时软件崩溃的问题
  • Ctrl/Cmd+W 现在可以关闭快速切换以及帮助窗口
  • 通过快捷键在文件列表中导航时现在会自动忽略隐藏文件
  • 在实时预览模式下点击表格或标注的编辑按钮时，现在视图会自动滚动到合适位置
  • 字数统计功能现在对于任何打开的纯文本文件都生效，而非只是 markdown 文件
